# Church Events App

## Overview
A web application to provide quick and readily available information about the Church year's events for members of the community.

## Project Structure
- `index.html` - Main HTML page (static site)
- `README.md` - Project description

## Current State
The project is set up as a static HTML site served via Python's built-in HTTP server for development. It is configured for static deployment.

## Development
The site runs on port 5000 using Python's http.server module.

## Deployment
Configured as a static site deployment serving from the root directory.
